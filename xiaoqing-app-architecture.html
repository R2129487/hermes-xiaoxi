小青 App — 系统架构

跨平台即时通讯 · 直连 Hermes Agent · 复用 MESH 能力层

什么是小青 App？

  • Flutter 跨平台 App（Android / iOS）
  • 替代微信/QQ 作为与小青团队的通信入口
  • 直连 MESH 服务器，不走第三方平台
  • 自由控制通信内容、文件大小、推送频率
  • 可备案为正式互联网应用

MESH 合并策略

  • App 注册为 MESH 特殊节点 "user"，不参与任务委派
  • 复用 MESH WebSocket 消息路由 + 离线存储
  • 复用 MESH 权限/认证系统（已有 token 机制）
  • 复用 MESH 状态管理 + 审计日志
  • MESH 侧新增：推送服务、文件上传接口

App 核心功能

  • 聊天界面：文字、图片、文件、语音消息
  • 任务列表：查看执行状态、结果、重试
  • 推送通知：后台消息提醒
  • 多 Agent 选择：直接发给小青/小蓝/小白
  • 文件传输：直接上传/下载中转文件
