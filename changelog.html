v0.1.2 最新 2026-06-27

✨ 新功能

  • 流式语音识别 — 基于 Zipformer-zh-int8 模型，边说边出字，告别等待
  • 系统通知 — 收到智能体回复时，状态栏弹出通知 + App图标角标
  • 用户间私聊 — 联系人列表点击用户头像，进入一对一私聊
  • 调度器用户路由 — 后端支持用户间消息转发，会话ID自动对齐

🔄 改进

  • 语音模型优化 — 从 Paraformer（606MB）迁移到 Zipformer-zh-int8（160MB），APK 体积可控
  • 语音按钮逻辑 — 麦克风和发送按钮解耦——录音中随时可发送，按钮始终可见
  • UI布局优化 — 注册表单、设置页保存按钮固定底部，不随内容滚动
  • 聊天文本累积 — 语音识别结果追加到输入框，不再每次覆盖

v0.1.1 2026-06-26

✨ 新功能

  • 用户认证系统 — JWT 登录/注册，bcrypt 密码加密，token 自动持久化
  • 三级权限体系 — admin / operator / observer 角色，注册自动降级防越权
  • 用户管理页面 — 管理员在设置页查看所有用户、修改角色
  • 联系人搜索 — 按名称/ID 实时过滤
  • APP内注册智能体 — 联系人页底部表单，支持声明能力模块
  • 暗色模式 — 全页面覆盖，一键切换
  • 用户删除 — 管理页可删除用户，需输入 OK 二次确认

🔄 改进

  • 仓库重构 — xiaoxi-mesh → hermes-xiaoxi，三模块合一
  • Flutter 3.44 — 全面兼容最新 SDK，移除弃用 API
  • WebSocket 通信 — 连接稳定性优化，断线重连机制
