核心卖点

桌面版 / 机器人版

🤖 双Agent协作架构

ESP-Claw（底层Agent）+ Hermes（高级大脑）

🦾 ESP-Claw — 感知、执行、本能反应（毫秒级）
🧠 Hermes — 推理、规划、记忆、创造（深度思考）
🔗 MCP协议 — 局域网内实时双向通信

💡 就像人的神经系统：脊髓管反射（不过脑子），大脑管思考。ESP-Claw是脊髓，Hermes是大脑。

GPIO/舵机控制 摄像头/传感器 LED/喇叭 深度推理 工具调用 长期记忆
笔形版

🖊️ 哑巴终端架构

ESP32-C3 纯透传 + Hermes 全权负责

🎤 ESP32-C3 — 只管收音和播放（纯透传）
🧠 Hermes — 全部智能（ASR+推理+TTS）
📡 HTTP — 音频流传输到服务器

💡 极致精简：C3芯片成本极低，体积小到能塞进笔杆。所有智能在云端，设备只做音频管道。

技术栈

🐾 ESP-Claw Agent

  • 乐鑫官方开源ESP32 AI Agent框架
  • 内置Lua脚本引擎控制硬件
  • 72个能力模块（19个技能）
  • 380个源文件，12.5万行代码
  • 固件大小仅3.4MB
  • 已在ESP32-S3-N16R8上运行

  • Model Context Protocol 标准协议
  • 设备既是Server又是Client
  • 局域网内低延迟通信

  • 工具发现与调用机制

快速开始

1

硬件准备

准备 ESP32-S3-N16R8 开发板、I2S 麦克风、扬声器模块

推荐硬件：ESP32-S3-DevKitC-1-N16R8 + INMP441麦克风 + MAX98357A功放
2

克隆ESP-Claw

获取乐鑫官方AI Agent框架

git clone --recursive https://github.com/espressif/esp-claw.git cd esp-claw
3

编译烧录

使用ESP-Claw的idf.py bmgr方式编译

idf.py set-target esp32s3 idf.py bmgr build # 使用构建管理器编译 idf.py -p /dev/ttyUSB0 flash monitor
4

配置Hermes连接

配置ESP-Claw连接Hermes高级大脑

# 在menuconfig中配置 idf.py menuconfig # 设置WiFi、Hermes服务器IP、MCP端口 # 启动Hermes后，对小希说 "你好小鑫" 开始对话
